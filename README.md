# DFusion-DFI

Data_process:

The original data is first processed by the code document "predata.py";



Execute files in different folders according to different datasets:

DDinter2.0:

For drug-food interaction prediction: Execute main_function.py

For DFI mechanism prediction: Execute main_function_mechanism.py

ArcDFI:

The ArcDFI dataset is derived from the paper ArcDFI:https://www.biorxiv.org/content/10.1101/2025.04.09.647981v1.full

ArcDFI Dataset:https://drive.google.com/file/d/1TAr6v1Swfn4F_O_GyKKQf6KlsWh8lQ5N/view

Execute main_function.py, and implement different partitions by modifying the value of the flag parameter in the file.
